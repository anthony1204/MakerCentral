instructions on how to use makercentral.io after it has been closed.

hello! i'm Anthony! i have found a way to access makercentral.io after it has been closed. here are the instructions:

first, go to my repl on replit.com and click the fork button and follow the instructions below.

to set it up, fork the repl, press Control+Shift+s, type cd 'MakerCentral-main', type 'npm install' then wait a while. then type 'npm run dev' once its done. to access it go to the website 'https://[your project name aka makercentral].[your replit username].repl.co' which in my case is 'https://makercentral.anthony1204.repl.co'. most likely wont let you open the level details but you can use wizulus's level viewer for that. just copy the id in the gray text there.
